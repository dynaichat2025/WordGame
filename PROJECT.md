작성일: 2026-03-05

## 1. 프로젝트 개요

**제품명:** Word Game (한글 단어 학습 퀴즈)
**목표:** 초등학교 5학년 이하(10세 이하) 어린이를 위한 한글 어휘력 향상 퀴즈 게임
**플랫폼:** 웹 브라우저 (React + Vite)
**배포:** Vercel 무료 배포 예정

---

## 2. 게임 개념

문장 안에 포함된 단어의 뜻을 4지선다로 고르는 퀴즈.
컨텍스트가 있는 완전한 문장을 제시하여 단어를 자연스럽게 학습.

**문제 형식 예시:**
```
문장: "할머니 댁은 허름한 골목 끝에 있었다."
질문: 위 문장에서 '허름한'의 뜻은 무엇인가요?

① 낡고 보잘것없는    ← 정답
② 밝고 화려한
③ 크고 넓은
④ 단단하고 무거운
```

---

## 3. 게임 규칙

- 4지선다 객관식 퀴즈
- 대상: 10세 이하, 초등학교 5학년 이하
- 문장 속 타깃 단어의 뜻을 고르는 방식
- 정답/오답 즉시 피드백
- 최종 점수 및 순위 기록

---

## 4. PRD 요약

### 사용자
- 주 사용자: 초등학교 1~5학년 (6~11세)
- 부 사용자: 학부모, 교사 (점수/기록 확인)

### 핵심 기능

**퀴즈 게임플레이**
- 컨텍스트가 있는 완전한 문장 제시
- 문장 안 타깃 단어 굵게/색상 강조 표시
- 초등 5학년 이하 한글 어휘 단어
- 회당 10문제 (랜덤 셔플)
- 문제당 제한 시간 (난이도별 상이)

**즉시 피드백**
- 정답: 초록색 하이라이트 + "정답! 🎉"
- 오답: 빨간색 하이라이트 + "오답! 정답: [뜻]"
- 1.5초 후 다음 문제 자동 이동

**점수 시스템**
- 기본 점수: 정답 10점
- 스피드 보너스: 남은 시간 x 0.5점
- 콤보 보너스: 연속 정답 3개부터 콤보 x 2점

**순위 기록 (리더보드)**
- localStorage에 저장
- 닉네임 + 점수 + 날짜 + 난이도
- 상위 10위 표시 / 개인 최고점 표시

### 난이도

| 레벨 | 대상 | 단어 수준 예시 | 제한 시간 |
|------|------|---------------|-----------|
| Easy | 1~2학년 | 가득, 소중한, 따뜻한 | 60초 |
| Normal | 3~4학년 | 부지런한, 조심스럽게, 넉넉한 | 60초 |
| Hard | 5학년 | 허름한, 드물다, 빽빽한 | 60초 |

### 화면 구성 (Flow)

```
시작 화면 (닉네임 입력 + 난이도 선택)
    └─ 퀴즈 화면 (문장 제시 → 선택 → 즉시 피드백 반복)
         └─ 결과 화면 (최종 점수 + 리더보드)
              └─ 다시 하기 / 처음으로
```

## 5. 기술 스택

| 항목 | 선택 |
|------|------|
| 프레임워크 | React + TypeScript |
| 빌드 도구 | Vite |
| 스타일 | Tailwind CSS |
| 데이터 저장 | localStorage |
| 배포 | Vercel (무료) |

### 폴더 구조

```
src/
  components/
    StartScreen.tsx
    QuizScreen.tsx
    ResultScreen.tsx
  data/
    questions.ts
  types/
    index.ts
  App.tsx
  main.tsx
  index.css
```

### 타입 정의

```ts
type Difficulty = 'easy' | 'normal' | 'hard'

interface Question {
  id: number
  sentence: string        // 타깃 단어가 포함된 완전한 문장
  word: string            // 문장 안의 타깃 단어 (강조 표시용)
  options: [string, string, string, string]
  answer: 0 | 1 | 2 | 3
  difficulty: Difficulty
}

interface PlayerRecord {
  nickname: string
  score: number
  correct: number
  total: number
  difficulty: Difficulty
  date: string
}
```

## 6. 구현 3단계 프롬프트

### Stage 1 — 프로젝트 기반 구축

- Vite + React + TypeScript + Tailwind 세팅
- 한글 어휘 문제 데이터 20개 작성
  - easy 8개 (1~2학년: 가득, 소중한, 따뜻한 등)
  - normal 8개 (3~4학년: 부지런한, 조심스럽게, 넉넉한 등)
  - hard 4개 (5학년: 허름한, 드물다, 빽빽한 등)
- 타입 정의
- StartScreen / QuizScreen / ResultScreen 기본 틀
- App.tsx 화면 전환 로직

### Stage 2 — 게임 로직 및 인터랙션 완성

- 타이머 + 진행바 UI (난이도별 20/15/10초)
- 즉시 피드백 (색상, 메시지, 1.5초 후 자동 이동)
- 점수 시스템 (기본 10점 + 스피드 보너스 + 콤보)
- 문장 안 타깃 단어 강조 표시 (굵게 + 색상)
- 애니메이션 (문제 전환, 정답 펄스 효과)

### Stage 3 — 완성도 및 UX 개선

- 리더보드 (상위 10위, 현재 플레이어 하이라이트)
- 결과 화면 (정확도, 등급 S/A/B/C)
- 키보드 단축키 (1/2/3/4 선택)
- 접근성 (큰 글씨 최소 18px, 고대비 색상)
- README.md 작성
